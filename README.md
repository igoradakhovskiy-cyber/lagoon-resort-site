# Lagoon Resort — Landing Pages

Прототип лендингов для проекта Lagoon Resort. Статический сайт (HTML / CSS / JS), без сборки.

## 🔗 Живая версия

После включения GitHub Pages сайт доступен по адресу:
**https://igoradakhovskiy-cyber.github.io/lagoon-resort-site/**

## Страницы

| Файл | Назначение |
|------|-----------|
| `index.html` | Главный лендинг (RU) |
| `lp-capital.html` | Лендинг под оффер «Capital» (RU) |
| `lp-income.html` | Лендинг под оффер «Income» (RU) |
| `en/index.html` | Английская версия |
| `en/lp-capital.html` | EN — Capital |
| `en/lp-income.html` | EN — Income |

## Структура

```
.
├── index.html            # лендинги (русская версия)
├── lp-capital.html
├── lp-income.html
├── en/                   # английская версия
├── assets/
│   ├── css/styles.css
│   ├── js/app.js
│   └── img/              # изображения
└── README.md
```

## Локальный запуск

```bash
python3 -m http.server 4321
# открыть http://127.0.0.1:4321
```
